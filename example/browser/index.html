Vue-Chimera

Easiest way to interact with RESTful APIs

Simple GET

{{ simpleGet.data.title }} Loading... 

chimera: {
    simpleGet: '/api/v1/posts/1'
}

Reactive GET

Post id:

{{ reactiveGet.data.title }} Loading... 

chimera: {
    reactiveGet() {
        return `/api/v1/posts/${this.postId}`
    },
}

Simple GET with auto refresh interval

Current UTC time: Last loaded: {{ time.lastLoaded }} Refreshing... 

chimera: {
    time: {
        url: '/api/v1/posts/1',
        interval: 2000
    }
}

POST with manual send

Successfully created: {{ manual.data.name }} {{ manual.data.job }}

Loading... 
chimera: {
    manual: {
        url: '/api/v1/posts/1',
        auto: false
    }
}
